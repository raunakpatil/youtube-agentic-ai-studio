# Contributing to YouTube AI Agent Studio

Thanks for your interest in contributing! Here's how to get involved.

## Reporting bugs

Open an [issue](https://github.com/yourusername/youtube-ai-agent/issues) with:
- Your OS and Python version
- The exact error message / traceback
- Steps to reproduce

## Suggesting features

Open an issue with the **enhancement** label. Describe what you'd like and why it would benefit users.

## Submitting a pull request

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes — keep them focused on a single improvement
4. Test locally: run `python pipeline.py` end-to-end at least once
5. Commit with a clear message: `git commit -m "Add: short description"`
6. Push and open a PR against `main`

## Code style

- Python: follow PEP 8, type hints encouraged
- Keep functions small and single-purpose
- Add a comment if the intent isn't immediately obvious
- Do not commit API keys, tokens, or any personal data

## Areas most in need of help

- Thumbnail generation (AI-designed cover art)
- A proper test suite
- Docker / one-click install packaging
- Non-English voice and channel support
- UI improvements to the GUI

All contributions, no matter how small, are appreciated.
